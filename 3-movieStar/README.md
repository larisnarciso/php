# Movie Star

![Movie Star](moviestar.jpg)

Projeto Movie Star, criado utilizando PHP e Design Patterns DAO. O usuário poderá realizar o cadastro no Movie Star, onde ele poderá adicionar e realizar reviews de filmes.

## CRUD

- Criar
- Ler
- Atualizar
- Deletar
